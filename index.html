Automatic APK detection

When an Android device opens install.cat/user/repo, we detect the device architecture and find the best APK from your latest GitHub release. Naming priority:

  1. {repo}-android-{arch}.apk (exact match)
  2. *android*{arch}*.apk (partial with arch)
  3. *android*.apk (any Android APK)
  4. *.apk (any APK)
